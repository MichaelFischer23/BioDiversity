Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

This project was created using a mixture of html, javascript, and plotly. The data can be sorted through using the drop down menu to select which individual's data you want. The demographic info box displays the subjects ID, eithnicity, gender, age, location, belly button type (inner/outer) and the number of times they wash their belly button in a given week. The bar and bubble charts displays the most common bacteria found in each persons belly button. While the guage chart displys how often it is washed.
